Popular Salads:
returns array of salads sorted by popularity

1) GET: /pantry/reports/popularSaladsByGender
   description: returns popular salads based on the gender of the users who placed orders
   inputs: 
      1. "gender" character possible inputs: 'M' or 'F'
	   2. limit*

2) GET: /pantry/reports/popularSaladsByAge
   description: returns popular salads based on the age-range of the users
   inputs: 
      1. "startAge" integer minimum age 
		2. "endAge" integer maximum age
		3. limit*

3) GET: /pantry/reports/popularSaladsByLocation
   description: returns popular salads based on the location given
   inputs:
      1. "location" latitude, longitude in Decimal degrees 
		2. limit*

4) GET: /pantry/reports/popularSaladsByTime
   description: returns popular salads based on the time of the day
   inputs:
      1. "startTime" String format: HH:mm:ss
		2. "endTime"   String format: HH:mm:ss
		3. limit*
